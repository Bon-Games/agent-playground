# GitHub Plugin (Claude.ai)

**This is a Claude.ai plugin — it cannot be installed via Docker.**

It requires manual authorization through your browser and is tied to your Claude account.

## Manual installation

1. Open https://claude.com/plugins/github in your browser.
2. Click **Connect** and authorize with your GitHub account.
3. The plugin will appear automatically in your Claude.ai sessions.

## Notes

- Works in Claude.ai web and Claude Code (via your account).
- Does not run locally — GitHub API calls are proxied through Anthropic.
- Not compatible with Gemini or other AI providers.
