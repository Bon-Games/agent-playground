# Windows MCP

Git source: https://github.com/CursorTouch/Windows-MCP

Provides Windows OS automation tools: app control, PowerShell, file system, screenshots, clicks, typing, and more.

## Platform constraint

**This server only works on Windows hosts.** It will be skipped automatically on macOS and Linux.

On a Windows machine, include `windows-mcp` in your profile and run `./install.sh` as normal.

## Requirements

- Windows OS
- Python 3.13+
- Docker Desktop for Windows
